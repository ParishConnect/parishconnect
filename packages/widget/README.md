# @parishconnect/widget

React-based widget for displaying and managing Catholic parish information and mass schedules.

## Development

```bash
# Start development server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview

# Lint code
bun run lint
```

## Dependencies

This package depends on:
- `@parishconnect/schema` - Data schemas and types
- `@parishconnect/rrule-converter` - Schedule conversion utilities

## Features

- Display parish information
- Mass schedule viewer
- Event management form
- Multi-language support (i18n)
- Theming support
- Generates JSON-LD for SEO
