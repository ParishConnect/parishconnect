## Installation

To add the HTML Mass Times Widget to your website, first check:

- Your website supports embedding external widgets (most do).
- You have access to edit the HTML of your website.

## Usage

```html
<script src="https://cdn.parishconnect.io/masstimes-widget.js"></script>
<parishconnect-masstimes></parishconnect-masstimes>
```
